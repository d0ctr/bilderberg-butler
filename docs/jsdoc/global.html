commonizeContext(ctx, limit) → {Interaction}

Turns Telegram context to an object that can be used as an input to a common command handler
Parameters:
Name Type Description
ctx Object
limit Integer number of parsable args
Source:
  • commands/telegram.js, line 21
Returns:
`limit` is tricky, it makes possible for argument to consist of multiple words Example: `/foo bar baz bax` - if we set limit here to 1, we will limit the number of args to 1 and this function will join all args with spaces, therefore args = ['bar baz bax']. - if we set limit to 2, we will have 2 args as follows: args = ['bar', 'baz bax'], and so on. - if we set limit to null, we will parse all words as standalone: args = ['bar', 'baz', 'bax'].
